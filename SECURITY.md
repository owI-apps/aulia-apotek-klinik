Security Policy
Supported Versions
Versi aplikasi yang saat ini mendapatkan dukungan pembaruan keamanan. Karena aplikasi ini baru diluncurkan, kami hanya mendukung versi terbaru.

Version
Supported
1.0.x	:white_check_mark:
< 1.0	:x:

Reporting a Vulnerability
Keamanan data pasien dan transaksi keuangan adalah prioritas utama kami. Jika Anda menemukan celah keamanan (vulnerability) pada sistem Aulia Apotek Klinik, mohon laporkan kepada kami dengan cara:

JANGAN membuka Issue publik di repository GitHub ini.
Kirim laporan detail via email ke: fciowi.aplikasi@gmail.com
Sertakan langkah-langkah reproduksi (cara menemukan celah tersebut) dan screenshot jika memungkinkan.
Apa yang akan terjadi selanjutnya?
Anda akan mendapatkan email konfirmasi penerimaan laporan dalam waktu maksimal 48 jam.
Kami akan menyelidiki laporan tersebut dan memberikan perkiraan waktu perbaikan dalam 3-5 hari kerja.
Jika kerentanan diterima (valid), kami akan segera merilis patch (perbaikan) dan mengucapkan terima kasih kepada Anda (credits) atas bantuan menjaga keamanan sistem ini (kecuali Anda memilih untuk anonim).
Jika kerentanan ditolak, kami akan memberikan penjelasan teknis mengapa hal tersebut tidak dianggap sebagai celah keamanan.
Terima kasih telah membantu menjaga keamanan Aulia Apotek Klinik.
